GUIDA COMANDI BASE GIT
--> CAMBIARE DIRECTORY: cd 'percorso file'
--> CONOSCERE LO STATO DELLA DIRECTORY: git status
	--> File in verde sono pronti per la commit e fanno parte della staging area, file in rosso non sono stati ancora aggiunti alla directory
	    e sono solamente in locale
--> AGGIUNGERE SINGOLO FILE A STAGING AREA: git add nomefile.estensione
--> AGGIUNGERE TUTTI I FILE A STAGING AREA: git add *
--> APRIRE EDITOR PER COMMIT: git commit
	--> cliccando 'i' posso inserire commenti e per uscire clicco esc e poi ':wq'
--> FARE LA COMMIT CON COMMENTO DA LINEA DI COMANDO: git commit -m "commento"
--> CANCELLARE PROGRESSI TORNANDO A ULTIMA COMMIT: git checkout -- 'nomefile'
--> VEDERE CHE BRANCHES CI SONO E SU QUALE SONO: git branch
--> CREARE UN BRANCH: git branch 'nome branch'
--> SPOSTARSI AD UN CERTO BRANCH: git checkout 'nome branch'
--> FARE IL MERGE DI UN BRANCH: git merge 'nome branch di cui fare il merge su quello in cui mi trovo'
